Local only

Portfolio project editor

This tool runs only on your computer. It edits data/projects.js, creates a local backup, and can publish through local git when you confirm it.

This editor is local only and binds only to 127.0.0.1.

It does not use GitHub tokens, passwords, external APIs, analytics, or remote uploads.

It can only write data/projects.js and data/projects.backup.js. Image files still need to be added manually to assets/projects/[slug]/.

Workflow

Edit, apply, inspect the image checks, save locally, preview the site, then publish when you are ready.
