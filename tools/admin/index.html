Local only

Portfolio project editor

This tool runs only on your computer. It edits data/projects.js, saves locally, and can publish through local git.

This editor is local only and binds only to 127.0.0.1.

It does not use GitHub tokens, passwords, external APIs, analytics, or remote uploads.

Image files still need to be added manually to assets/projects/[slug]/.

Workflow

Edit, apply, save locally, then publish when you are ready.
